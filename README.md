# Clothing Website Project

## 📖 Project Description
This project is an online clothing store developed as a university project.

The website allows users to browse clothing products, add items to cart, and manage purchases.

The backend is connected to database services to handle user authentication and product data.

## 🎯 Purpose of Project
- Learn full stack web development
- Practice frontend and backend integration
- Understand e-commerce system design

## 💡 Features
- User registration and login system  
- Product listing and browsing  
- Shopping cart management  
- Admin panel for product control  
 

## 🛠 Technologies Used
- React.js (Frontend)
- Node.js (Backend)
- HTML, CSS, JavaScript
- Database System (MySQL )

## 📌 Repository
Hosted on GitHub platform.

## 🚀 How to Run Project
1. Clone repository
2. Install dependencies
3. Run frontend server
4. Run backend server
